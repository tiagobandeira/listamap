# listamapp
Development project of the Neutron Group
### Prerequisites

What things you need to install the software and how to install them

```
npm  : 5.5.1 
Ionic (Ionic CLI) : 3.19.1
Node : v8.9.1
```
### Installing

Install listamapp

```
$npm install
```

Add platform and run

```
$ionic cordova run android
```
